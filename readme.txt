== Description ==

This sidebar widget provides a list of authors.

== Installation ==

1. Ensure that your theme is <a href="http://codex.wordpress.org/Theme_List/Widget-ready">widget-ready</a>.
1. Ensure you are running WordPress-2.2 or later. The "Authors" widget already exists in the <a href="http://automattic.com/code/widgets/">Widgets plugin</a> for WordPress-2.1.
1. Upload `wp-authors.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to the 'Presentation' menu, click on 'Widgets', and add the 'Authors' widget to the sidebar.

== Frequently Asked Questions ==

= Isn't this already a widget? =

Yes, it is, in the older <a href="http://automattic.com/code/widgets/">widgets plugin</a> releases. For some reason, "Authors" was removed for WordPress-2.2.
